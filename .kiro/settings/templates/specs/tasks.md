## JIRAラベル一覧

各ストーリーには以下のラベルが自動付与されます：

- `Requirements` - 要件定義フェーズ
- `Design` - 設計フェーズ
- `Implementation` - 実装フェーズ
- `Testing` - 試験フェーズ
- `Release-Preparation` - リリース準備フェーズ
- `Release` - リリースフェーズ

**JIRAフィルター例**:
```text
project = PROJ AND labels = "Requirements"    # 要件定義タスクのみ
project = PROJ AND labels = "Testing"         # 試験タスクのみ
project = PROJ AND labels = "Release"         # リリースタスクのみ
```

**次のステップ**: 
1. tasks.mdを上記テンプレートに従って作成

3. JIRAに全フェーズのストーリーが自動作成される
