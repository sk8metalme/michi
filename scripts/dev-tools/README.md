# Development Tools (開発ツール)

## 目的

開発時に利用する補助ツールを配置。プロダクションビルドには含まれません。

## 配置するもの

- **test-interactive.ts**: 対話型テスト実行ツール
- **mermaid-converter.ts**: Mermaid図の変換ツール
- その他の開発補助スクリプト

## 実行タイミング

- 開発者が手動で実行
- CI/CDの一部として実行（テスト等）

## 注意事項

- このディレクトリのスクリプトはプロダクションビルドに含まれない
- プロダクションコードは `src/` 配下に配置
- ビルドスクリプトは `scripts/build/` に配置
